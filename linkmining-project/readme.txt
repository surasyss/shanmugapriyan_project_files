
Install
--------
cd project_directory
npm Install

BUILD
------
npm run watch:dev


Above command will generate the build directory,

How to install the extension
Steps:
1/ Point the build directory to chorme by following below steps
2/ go to chrome browser extension window
3/ enable developer option(present in right side of the screen)
4/ then you will get option called load unpacked.
5/ select and load build directory.
6/ Now extension installed in your chrome browser.


How to work with extension:

1/ First click the orange extension icon
2/ then click dashboard
3/ then click start indexing
4/ it will show the indexing status in dashboard
5/ once the indexing completed you can go select results to see related links
